Credit Card App UX Sprint

Team: Just Me
Role: UX Designer
Timeline: 2 weeks
Scale: Thousands of users
Technology: Figma

Problem: MVP features for a new financial product and eventual white-label credit card app were in flight, and release 2 features were yet to have thinking or requirements. New to the team and unfamiliar with the product, I was tasked with thinking through the features and creating some user journeys and sample flows over the holiday break.

Process:

  • Using the experience canvas method, I wrote some storyboards for different users based on the planned feature set, existing features, and strategy document that identified target users
  • Conducted casual competitive analysis of features and flows to better understand how we could innovate
  • Distilled screen flows from user journey storyboards
  • Wirframed the flows using existing visual components, proposing features and integrations along the way

Outcome: Feature outlines and thinking were utilized throughout the design. New feature ideas for the design were pitched to the team and added to our backlog
